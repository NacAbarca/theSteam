The Steam is a responsive Wordpress theme specially designed for restaurants, pubs and alike.

It features a single layered frontpage menu along four main dish categories which can be customized to suit
your special needs. Each dish is presented beautifully in a multi-page menu book, also accessible in the main page. 

For more information please consult the help file from the docs folder or visit our website: www.webdotinc.net / www.webdotinc.com
